## üíª Quick Installation

1. **Clone the repository:**
```bash
git clone https://github.com/AndyAlvarezC/portfolio.git
````

2. **Enter the project folder:**
```bash
cd portfolio
`````

3. **Install dependencies**
```bash
npm install
`````

4. **Start development server:**
```bash
npm run dev
`````

5. **Open in browser:**
```bash
http://localhost:5173
````
